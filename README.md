# VX Launcher — Portable Minecraft Launcher

[![Release](https://img.shields.io/github/v/release/VishvaGIT-hub/VXLauncher?color=purple&label=Release)](https://github.com/VishvaGIT-hub/VXLauncher)
[![License](https://img.shields.io/github/license/VishvaGIT-hub/VXLauncher?color=blue)](LICENSE)
[![Platform](https://img.shields.io/badge/Platform-Windows-blueviolet)](https://github.com/VishvaGIT-hub/VXLauncher)

A lightweight, portable Minecraft Java Edition launcher designed for speed, transparency, and convenience. Extract, launch, and play Minecraft instantly — no installation or administrative privileges required.

## Key Features

- 📦 **Fully Portable:** Run the launcher directly from a USB stick or any local directory. Zero registry keys created or background installers run.
- ⚡ **Fast Launch:** Optimized startup times to get you into the game in seconds.
- 🔧 **Mod Support:** One-click installations for Forge, Fabric, Quilt, and OptiFine loaders.
- 🎮 **All Java Versions:** Full support for every Java Edition release, from classic legacy builds to the latest snapshots.
- 🛡️ **Open Source & Secure:** Transparent open-source codebase with absolutely noTelemetry or spyware. Your authentication data stays local on your system.
- ☕ **Self-Managed Java:** The launcher automatically fetches, configures, and manages the correct Java Runtime environments for each Minecraft version.

## Quick Start

1. Go to the [Releases](https://github.com/VishvaGIT-hub/VXLauncher/releases) page.
2. Download the `VXLauncher-Portable-v1.1.4.zip` bundle.
3. Extract the ZIP file contents to any folder of your choice (e.g., your Desktop or a USB drive).
4. Run `VXLauncher.exe`.
5. Enter your account credentials, choose your desired Minecraft version, and hit **PLAY**.

## Repository Structure

- `index.html` - The public landing page.
- `style.css` - Custom style specifications for the landing page.
- `script.js` - Client-side scrolling, animations, and navbar controls.
- `assets/` - Logo assets and landing page artwork.
- `VXLauncher-Portable 1.1.4/` - Portable launcher executables and resources.
